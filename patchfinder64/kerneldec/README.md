# kerneldec
iOS Kernel Decompressor
```
Usage: kerneldec [OPTIONS]
        -h, --help         Print this help
        -d, --debug        increment debug level
        -i, --input NAME   Input from NAME instead of stdin
        -o, --output NAME  Output to NAME instead of stdout
        -k, --kpp NAME     Save KPP to NAME
        -q, --quiet        No non-error output
```
